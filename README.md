## –ò—Å—Ö–æ–¥–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ:

$$\iint^{}_{\Omega} h(x, y)dx dy,$$ –≥–¥–µ –æ–±–ª–∞—Å—Ç—å $\Omega = {(x, y): a \leq x \leq b \leq y \leq g(x)}$ —Å –Ω–µ–∫–æ—Ç–æ—Ä–æ–π —Ñ—É–Ω–∫—Ü–∏–µ–π $g$.
$a = 2; \\ b = 4; \\ g(x) = \sqrt{x-1}; \\ h(x, y) = \frac{sin(x+y) + cos(x-y)}{(x+y)}$.

## –•–æ–¥ —Ä–∞–±–æ—Ç—ã:

$$\iint^{}_{\Omega}{ h(x, y)dx dy} = S_{\Omega}\iint^{}_{\Omega}{ h(x, y) \frac{1}{S_{\Omega}} I_{\Omega} dx dy} $$

–ø–ª–æ—Ç–Ω–æ—Å—Ç–∏ —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ–≥–æ –Ω–∞ $|omega$ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è. –î—Ä—É–≥–∏–º–∏ —Å–ª–æ–≤–∞–º–∏, –∏–Ω—Ç–µ–≥—Ä–∞–ª —Å–æ–≤–ø–∞–¥–∞–µ—Ç
(—Å —Ç–æ—á–Ω–æ—Å—Ç—å—é –¥–æ –º–Ω–æ–∂–∏—Ç–µ–ª—è $S_{\Omega}$ ) —Å –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏–º –æ–∂–∏–¥–∞–Ω–∏–µ–º –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ

#### –¢–µ–æ—Ä–µ–º–∞

$$\frac{1}{n}\sum^{n}_{i = 1}{h(\xi_{i}, \eta_{i})} \rightarrow^{P}  J,$$

–≥–¥–µ $\rightarrow^{P}$ ‚Äì —Å—Ö–æ–¥–∏–º–æ—Å—Ç—å –ø–æ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏.

–ø—Ä—è–º–æ—É–≥–æ–ª—å–Ω–∏–∫–æ–º.

–ø–∞—Ä–∞–ª–ª–µ–ª—å–Ω—ã–º–∏ –æ—Å—è–º), –º–æ–∂–Ω–æ –∑–∞–¥–∞—Ç—å —Å –ø–æ–º–æ—â—å—é —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ–≥–æ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è
